# Research Guidelines (Android/Compose/LSP)

## Golden Rule
If unsure → research first. Do not guess.

## Source Priority
1. Official docs:
   - Android Developers
   - Kotlin docs
   - Jetpack Compose docs
2. LSP specification:
   - Microsoft LSP spec
   - JSON-RPC spec
3. Source code:
   - AOSP / Jetpack libraries
   - Reference implementations (read-only)

## Research Triggers
- Compose performance & recomposition
- Text editor data structures (rope/piece table)
- LSP message formats & edge cases
- Android process management for terminal/execution sandbox

## Documentation Output
Any risky decision must be recorded in planning as a decision log.
